Certification JavaScript

Dans cette formation, j'ai appris comment utiliser JavaScript, le langage de programmation dynamique utilisé pour rendre les sites web interactifs et dynamiques.

" Formation Open ClassRooms "

Lien internet : https://openclassrooms.com/fr/courses/7269381-apprenez-a-programmer-en-javascript

Mes objectifs

  1. Apprendre la syntaxe de JavaScript
  2. Manipuler les variables et les types de données
  3. Gérer les conditions et les boucles
  4. Créer des fonctions et des événements
  5. Manipuler le DOM avec JavaScript
  6. Déboguer et tester mon code JavaScript
  7. Comprendre les bases de la programmation asynchrone

Pourquoi cette formation ?

  1. Compléter ma formation en BTS SIO avec des compétences pratiques
  2. Utiliser JavaScript lors de mon stage et dans mon futur professionnel
  3. Acquérir des compétences utiles pour le développement web dynamique

Les outils que j'ai utilisé

  1. Navigateur Web (Google Chrome, Firefox)
  2. IDE : Visual Studio Code
  3. Console du navigateur pour déboguer
